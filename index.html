Patrón Composición

Permite componer objetos en estructuras de árbol para representar jerarquías parte-todo. Los clientes tratan objetos individuales y composiciones de manera uniforme.

¿Qué es?

El patrón define una interfaz Componente común para hojas (Leaf) y contenedores (Composite). Ambos responden a los mismos métodos, así el cliente no distingue entre ellos.

Participantes

Componente — interfaz base
Hoja — nodo sin hijos (archivo)
Compuesto — nodo con hijos (carpeta)

¿Cuando lo poedemos usar?

Cuando necesitamos representar jerarquias parte-todo: sistemas de archivos, menús anidados, organigramas, escenas 3D, etc.

// ─── Componente base (interfaz comun) ───────────────────────────────
class FileSystemNode {
  constructor(name) { this.name = name; }
  getSize()  { throw new Error('abstracto'); }
  display(indent = '') { throw new Error('abstracto'); }
}

// ─── Hoja: no tiene hijos ────────────────────────────────────────────
class File extends FileSystemNode {
  constructor(name, size) { super(name); this.size = size; }
  getSize()               { return this.size; }
  display(indent = '')    { console.log(`${indent}📄 ${this.name} (${this.size}KB)`); }
}

// ─── Compuesto: puede contener hojas u otros compuestos ─────────────
class Folder extends FileSystemNode {
  constructor(name) { super(name); this.children = []; }

  add(node)    { this.children.push(node); }
  remove(node) { this.children = this.children.filter(c => c !== node); }

  // getSize() delega recursivamente a sus hijos — uniforme!
  getSize() { return this.children.reduce((s, c) => s + c.getSize(), 0); }

  display(indent = '') {
    console.log(`${indent}📁 ${this.name}/`);
    this.children.forEach(c => c.display(indent + '  '));
  }
}

// ─── El cliente usa la MISMA interfaz para ambos ─────────────────────
const root = new Folder('proyecto');
const src  = new Folder('src');
src.add(new File('index.js',  12));
src.add(new File('app.js',    34));
root.add(src);
root.add(new File('README.md', 2));

root.display();
console.log('Tamaño total:', root.getSize(), 'KB'); // → 48 KB

from abc import ABC, abstractmethod

# ─── Componente base ─────────────────────────────────────────────────
class FileSystemNode(ABC):
    def __init__(self, name: str):
        self.name = name

    @abstractmethod
    def get_size(self) -> int: ...

    @abstractmethod
    def display(self, indent: str = ""): ...

# ─── Hoja ────────────────────────────────────────────────────────────
class File(FileSystemNode):
    def __init__(self, name: str, size: int):
        super().__init__(name)
        self.size = size

    def get_size(self): return self.size
    def display(self, indent=""):
        print(f"{indent}📄 {self.name} ({self.size}KB)")

# ─── Compuesto ───────────────────────────────────────────────────────
class Folder(FileSystemNode):
    def __init__(self, name: str):
        super().__init__(name)
        self._children: list[FileSystemNode] = []

    def add(self, node): self._children.append(node)
    def remove(self, node): self._children.remove(node)

    def get_size(self):
        return sum(c.get_size() for c in self._children)

    def display(self, indent=""):
        print(f"{indent}📁 {self.name}/")
        for child in self._children:
            child.display(indent + "  ")
